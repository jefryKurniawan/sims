# To Issues (Break Plan into Trackable Items)

**When:** Plan written, need to track progress.

## Method:

- Each plan step → GitHub Issue (or TASKS.md checkbox)
- Title: `[Module] Step description`
- Labels: `backend`/`frontend`/`db`/`test`
- Assignee: self
- Link PR to issue

## Minimal (no GitHub):

- Add to `TASKS.md` or lean-prd.md as sub-items
- Check off as done
